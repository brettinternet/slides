# Mermaid diagrams

Use the `mermaid` shortcode if you need to enable reveal-js highligting module
(`codeFences = false`). Otherwize, when `codeFences = true`, ` ```mermaid ``` ` codeblocks
are also supported. [More info](hugo-hl-example/#/5).

```code
{{</* mermaid */>}}
graph LR
    A --> B
    B --> C
{{</* /mermaid */>}}
```

results in:

{{< mermaid >}}
graph LR
    A --> B
    B --> C
{{< /mermaid >}}
